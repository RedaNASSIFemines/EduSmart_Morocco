QCM Questions

Question 1:
Écrire en utilisant des quantificateurs la phrase suivante : 'Tous les réels sont supérieurs à -1'.
A - $\forall x \in \mathbb{R}, x > -1$
B - $\exists x \in \mathbb{R}, x > -1$
C - $\forall x \in \mathbb{R}, x \geq -1$
D - $\exists x \in \mathbb{R}, x \geq -1$
Correct Answer: A
Question 2:
Quelle est la forme d'écriture correcte pour la phrase 'Il y a au moins un entier pair' ?
A - $\forall n \in \mathbb{Z}, n \text{ est pair}$
B - $\exists n \in \mathbb{Z}, n \text{ est pair}$
C - $\exists n \in \mathbb{Z}, n \geq 2$
D - $\forall n \in \mathbb{Z}, n \text{ est impair}$
Correct Answer: B
Question 3:
Quelle inéquation représente le fait que la somme de deux réels est toujours supérieure ou égale à l'un des deux ?
A - $x + y \geq x$
B - x + y \leq x
C - $x + y \geq 0$
D - $x + y \geq y$
Correct Answer: A
Question 4:
Écrire en utilisant des quantificateurs : 'Il existe un réel qui n'est pas un multiple d'un entier'.
A - $\exists x \in \mathbb{R}, \forall n \in \mathbb{Z}, x \neq nx$
B - $\forall x \in \mathbb{R}, \exists n \in \mathbb{Z}, x = nx$
C - $\exists x \in \mathbb{R}, \exists n \in \mathbb{Z}, x \neq nx$
D - $\forall n \in \mathbb{Z}, \exists x \in \mathbb{R}, x = nx$
Correct Answer: C
Question 5:
Quelle est la forme correcte du théorème du milieu pour les réels ?
A - $\forall a, b \in \mathbb{R}, a < b \Rightarrow \exists q \in \mathbb{Q}, a < q < b$
B - $\exists a, b \in \mathbb{R}, a < b \Rightarrow \exists q \in \mathbb{Q}, a < q < b$
C - $\forall a \in \mathbb{R}, a \neq 0 \Rightarrow \exists q \in \mathbb{Q}, a < q$
D - $\exists a \in \mathbb{R}, \forall b \in \mathbb{R}, b < a \Rightarrow \exists q \in \mathbb{Q}, b < q$
Correct Answer: A
